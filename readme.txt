This is a branch of dedart that expands it into a complete proof-of-concept mecab clone (prediction only, no training).

main.cpp is attributed to the public domain through the CC0 public domain attribution, and Unlicense.

Note: Micropather has its own license. You can replace it with any pathfinding library that supports unidirectional graphs.

g++ --std=c++17 main.cpp micropather.cpp -Wall -Wextra -pedantic -O3 -o dedart[.exe]
